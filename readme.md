## javascript线程

我觉得在开始描述相关问题之前，需要理解一下javascript里面的线程概念，首先需要知道：

*  javascript是单线程的，也就是说，一段代码，js执行的时候是从上往下一句一句的执行，前面的代码永远要先于后面的代码执行，如：

``` javascript
	var a = 15;
	var b = 16;
	//这里js代码在运行的时候，肯定先执行把15赋值给a的操作，再来执行把16赋值给b的操作
```
